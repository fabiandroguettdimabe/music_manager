Configuración de Cuenta YouTube Music

Para cargar tus playlists personales y tu lista de "Canciones que te gustan", necesitas configurar la autenticación de YouTube Music.

¿Cómo obtener las credenciales?

  1. Abre tu terminal y ejecuta: ytmusicapi oauth
  2. Sigue las instrucciones en pantalla para iniciar sesión en tu cuenta de Google.
  3. Copia el archivo JSON resultante (oauth.json) o su contenido de texto.
  4. Pega el contenido JSON a continuación.
